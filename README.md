# HANNA Studio — Brand System

Sistema de identidad visual para **HANNA Studio**: archivos de logo, isologo y documentación de fundamentos de marca.

---

## Estructura del repositorio

```
hanna-studio-brand-system/
├── assets/
│   └── logos/
│       ├── logo.svg        # Logotipo principal (HANNA STUDIO)
│       └── isologo.svg     # Isologo / monograma H
└── docs/
    └── brand-foundations.md  # Fundamentos de marca v1
```

---

## Archivos

### `assets/logos/logo.svg`

Logotipo principal del estudio. Combina:

- **HANNA** — tipografía Bodoni Moda (serif de alto contraste, estilo editorial)
- **STUDIO** — tipografía Montserrat SemiBold (descriptor secundario)

Dimensiones: `1200 × 600 px`

Paleta utilizada:

| Rol | Color | HEX |
|---|---|---|
| Fondo | Marfil suave | `#F8F4F2` |
| Texto principal | Grafito elegante | `#2F2B34` |
| Descriptor | Rosa acento | `#C899B3` |
| Detalle | Rosa nude | `#E8DCDC` |

---

### `assets/logos/isologo.svg`

Isologo de uso secundario. Contiene:

- Monograma **H** en Bodoni Moda sobre fondo marfil
- Marco circular en rosa nude como elemento gráfico de encuadre

Dimensiones: `512 × 512 px` (formato cuadrado, apto para avatares e íconos)

---

### `docs/brand-foundations.md`

Documento de fundamentos de marca v1. Incluye:

- Identidad inicial y valores de marca
- Sistema tipográfico (HANNA + STUDIO)
- Jerarquía visual
- Paleta de colores completa con valores HEX, RGB, CMYK y Pantone
- Reglas de uso de color (60 / 25 / 15)
- Próximos pasos para v2
