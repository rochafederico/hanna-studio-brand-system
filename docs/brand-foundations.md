# HANNA Studio — Brand Foundations v1

## 1. Identidad inicial de marca

Este documento define las bases iniciales de identidad visual para **HANNA Studio**.

La propuesta busca transmitir una marca:

* elegante
* femenina
* premium
* moderna
* fácilmente escalable a entornos digitales

---

## 2. Sistema tipográfico

### Logotipo principal — HANNA

Para la palabra **HANNA** se define una tipografía serif de alto contraste con estilo editorial y premium.

Se adopta la **opción “A”** validada en la propuesta visual.

### Tipografía recomendada

**Bodoni Moda**
*Alternativa compatible: Didot*

### Estilo

* uppercase
* alto contraste
* estilo fashion / beauty
* fuerte presencia visual
* personalidad distintiva en la letra **A**

Ejemplo:

```text
HANNA
```

---

### Descriptor — STUDIO

La palabra **STUDIO** debe utilizar la tipografía de comunicación y UI.

### Fuente

**Montserrat**

### Pesos recomendados

* Medium
* SemiBold

Ejemplo:

```text
STUDIO
```

Uso recomendado:

* tracking ligeramente expandido
* color de acento de marca

---

## 3. Jerarquía visual

```text
HANNA
STUDIO
```

Reglas:

* **HANNA** como elemento principal
* **STUDIO** como descriptor secundario
* mantener separación vertical consistente
* alineación centrada


Paleta de colores:

1. Grafito elegante

   * HEX: #2F2B34
   * RGB: 47, 43, 52
   * CMYK: 10, 17, 0, 80
   * Pantone: P 179-16 C

2. Rosa nude premium

   * HEX: #E8DCDC
   * RGB: 232, 220, 220
   * CMYK: 0, 5, 5, 9
   * Pantone: 7604 C

3. Rosa acento

   * HEX: #C899B3
   * RGB: 200, 153, 179
   * CMYK: 0, 24, 10, 22
   * Pantone: 7431 C

4. Marfil suave

   * HEX: #F8F4F2
   * RGB: 248, 244, 242
   * CMYK: 0, 2, 2, 3
   * Pantone: Warm Gray 1 C

Reglas de uso:

* 60% fondos claros
* 25% grafito
* 15% rosa acento
